# Ask AI 검색 정확도 개선

## 배경

기존 Ask AI는 비어 있는 벡터 저장소를 검색한 뒤 단순 단어 출현 횟수에 의존해, 표현이 다른 관련 문서를 놓칠 수 있었습니다.

## 변경점

- `index.json`의 제목과 AI 요약을 먼저 검색합니다.
- 직접 일치가 약할 때 로컬 임베딩 의미 검색을 실행합니다.
- 두 검색 순위를 합쳐 최대 3개 문서를 답변 근거로 사용합니다.
- 질문과 문서를 E5 모델 권장 형식으로 구분합니다.

## 결과

명확한 질문은 빠르게 찾고, 단어가 달라도 의미가 가까운 문서는 보조 검색으로 찾을 수 있습니다.
